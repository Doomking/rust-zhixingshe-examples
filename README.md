## 💡 愿景 (Mission Statement)

本仓库是 **Rust 知行社** 系列文章 **“Rust 具身智能实战”** 的官方代码示例集。

我们坚信 **Rust** 是下一代系统级 AI 和机器人应用的核心。本仓库旨在打破 Rust、JavaScript/TypeScript 与 AI/具身智能之间的技术壁垒，通过**实战代码**，帮助开发者实现 **知行合一**。

🚀 目标：实现基于 Rust 的高性能、低延迟的全栈具身智能解决方案。

## 🔬 技术栈与核心聚焦

| 领域 | 核心技术 | 重点 |
| :--- | :--- | :--- |
| **系统核心** | **Rust** (Cargo, Crates) | 安全、高性能、无畏并发 |
| **具身智能** | **dora-rs** (Dataflow) | 机器人操作系统、零拷贝通信 |
| **AI/ML** | **Candle**, **Tract** | 纯 Rust 高速推理、模型部署 |
